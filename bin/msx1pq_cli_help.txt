MMSXX - MSX1 Palette Quantizer
Usage: platform/Linux/bin/msx1pq_cli --input <file|dir> --output <dir> [options]
Usage (multiple inputs): platform/Linux/bin/msx1pq_cli --inputs <files...> --output <dir> [options]
Convert a single image or multiple images in a folder into images that comply with MSX1 (TMS9918) display rules.
Options:
  --input, -i <file|dir>       Specify the input PNG file or directory
  --inputs <files...>         Specify multiple input PNG files (all options are shared)
  --output, -o <dir>           Specify the output directory
  --out-prefix <string>       Prefix to add to output file names
  --out-suffix <string>       Suffix to add before the output file extension
  --out-sc2                   Output SCREEN2 .sc2 binary
  --color-system <msx1|msx2>   (default: msx1)
  --dither / --no-dither       (default: dither)
  --palette92                  (for dev) Output 92 color palette without dithering
  --dark-dither / --no-dark-dither (default: use dark dither palettes)
  --no-preprocess             Skip preprocessing adjustments
  --8dot <none|fast|basic|best|best-attr|best-trans> (default: best)
  --distance <rgb|hsv>         (default: rgb)
  --weight-h <0-1> --weight-s <0-1> --weight-v <0-1>
  --weight-r <0-1> --weight-g <0-1> --weight-b <0-1>
  --pre-posterize <0-255>      Apply posterization before processing (default: 16,  skipped if <= 1)
  --pre-sat <0-10>             Increase saturation before processing (default: 0.0)
  --pre-gamma <0-10>           Apply a gamma curve before processing (default: 1.0)
  --pre-contrast <0-10>        Adjust contrast before processing (default: 1.0)
  --pre-hue <-180-180>         Adjust hue before processing (default: 0.0)
  --disable-colors <index|range>... Disable palette indices (1-15). e.g. --disable-colors 2 4 7-8 15. At least two colors must remain enabled
  --pre-lut <file>             Apply RGB LUT (256 rows) or .cube 3D LUT before processing
  -f, --force                  Overwrite without confirmation
  -v, --version                Show version information
  -h, --help                   Show usage based on locale (Japanese if detected)
  --help-ja                    この日本語のUSAGEを表示
  --help-en                    Show this usage in English
